# Zorvyn Frontend

React and Vite frontend for the finance dashboard assignment.

## Features

- Login screen using backend seeded credentials
- Role aware dashboard view
- Summary metrics (income, expense, net balance)
- Category totals and recent activity
- Transaction table for ADMIN and ANALYST roles

## Setup

1. Copy `.env.example` to `.env`.
2. Ensure `VITE_API_BASE_URL` points to backend API.
3. Install dependencies:

npm install

4. Run dev server:

npm run dev

Default app URL: http://localhost:5173
